#### Code Refactoring

- 아침에 Activity할 때의 코드 + 오늘 배운 시멘틱한 요소 
- 질문 : `<header>`요소는 섹션이 아니기 때문에 내부에 `<nav>`가 오지 않아야 한다고 이해했습니다. <br>
그래서 `<div class="header">`로 코딩하였는데요, 하단에는 `<footer>`요소를 써보았습니다. <br>
`<header>`와 `<footer>`가 한페이지에 공존해야 하는지 궁금합니다. 아니면 저처럼 써도 되는지요~
